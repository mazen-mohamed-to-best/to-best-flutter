Place the following font files in this directory before building:

Cairo family (Arabic):
  Cairo-Regular.ttf   (weight 400)
  Cairo-Medium.ttf    (weight 500)
  Cairo-SemiBold.ttf  (weight 600)
  Cairo-Bold.ttf      (weight 700)

Poppins family (English):
  Poppins-Regular.ttf   (weight 400)
  Poppins-Medium.ttf    (weight 500)
  Poppins-SemiBold.ttf  (weight 600)
  Poppins-Bold.ttf      (weight 700)

Download from Google Fonts:
  https://fonts.google.com/specimen/Cairo
  https://fonts.google.com/specimen/Poppins

The codemagic.yaml pre-build script downloads them automatically at CI time.
